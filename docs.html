set:

Sets line

Example: set line


jmp:

Jumps lines

Example: jmp lines


com:

Compare two values. If they are not equall skip next line.

Example: com value1 value2


mov:

Move a value to a register.

Example: mov value register


r!:

Get value from register

Example: com r!3 5


add:

Add. Add goes to register 0.

Example: add number1 number2


Subtract:

Sub. Sub goes to register 1. (number1-number2)

Example: sub number1 number2


mult:

Multiply. Multiply goes to register 2.

Example: mult number1 number2


div:

Divide. Divide goes to register 3. (number1/number2)

Example: div number1 number2
